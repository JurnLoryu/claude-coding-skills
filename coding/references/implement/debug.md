# 调试（出错就进这套，别加新功能）

- **STOP-the-Line**：异常立即停，存错误输出/日志/复现步骤。
- 六步按序（跳步即失效）：**复现 → 定位(哪层) → 缩小(最小失败例) → 修根因(不是症状) → 写回归测试 → 全套验证**。
- 症状 vs 根因：列表重复→UI 里 dedup(坏)；根因是 SQL JOIN 出重复→改查询(好)。一直问"为什么"。
- 非复现 bug：加时间戳日志缩窗口 / 人为延迟触发竞态 / 对比环境差异。
- 神器：`git bisect`（`start` → 标 bad/good → `run npm test`）自动定位肇事 commit。
