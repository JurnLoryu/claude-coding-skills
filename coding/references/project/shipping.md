# 上线（可逆、可观测、有计划）

- 上线前清单：代码(测试全过/无 TODO/无 console.log) + 安全(无密钥/audit 清/输入验证/限速) + 性能(CWV 达标/无 N+1/图片优化)。
- **功能开关全生命周期**：上线(off) → 内部 on → 5% → 25% → 50% → 100% → 2 周内清 flag 和死代码；每个 flag 标所有者 + 过期日。
- **灰度阈值表**：错误率基线±10% 继续 / 10–100% 查 / >2× 立即回滚；P95 同理。
- 监控三类：应用(错误率/延迟/量/业务) + 基础(CPU/内存/连接池) + 客户端(CWV/JS 错误)。
- **回滚计划三件套**：何时触发(条件) / 怎么滚(开关<1min、重部署<5min、DB<15min) / 谁通知。
